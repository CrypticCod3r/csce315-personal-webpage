For my ECEN 350 class I took in the 2019 Fall Semester, I had the opportunity to work on a very cool project. I was tasked with programming a single-cycle processor which was fully capable of executing assembly instructions in machine language. Not only was this project super exciting when it finally worked, but I also had the chance to learn so much from it. This project gave me a deeper appreciation for the design of processors and computing systems. Furthermore, this class project solidified my understanding of both machine language and Verilog.

Spring semester of Freshman year, 2018, I got to work on the project that was known as the bane of freshman Aggie engineering students - the Marble Sorter. My introductory engineering classes taught me programming primarily through MATLAB, but also using LabVIEW. Sadly for us students, we had to do program this project using LabVIEW, and build it using styrofoam and popsicle sticks. Besides the impractical resources we had at our disposal, we also had to figure out how to get together a team of 8 students, and plan and execute a project that would take half a semester. This project was both very difficult and very fun, and I had the opportunity to learn so much about communication and teamwork.
